# Contributing

First of all, thank you for your interest in contributing to **System Update Manager**.

## Reporting Bugs

Please include:

- Linux distribution
- Distribution version
- Kernel version
- Filesystem
- Full error message
- Steps to reproduce the issue

## Feature Requests

Suggestions are always welcome.

Examples:

- Support for additional Linux distributions
- New backup engines
- Better rollback functionality
- Performance improvements
- Security enhancements

## Pull Requests

Before submitting a Pull Request please:

- Test your changes.
- Keep the code readable.
- Follow the current coding style.
- Update the documentation if required.
- Update CHANGELOG.md if appropriate.

Thank you for helping improve this project.
